CNC-BOT DIBUJANTE

Integrantes:

Juan David Ortiz Muñoz

John Fernando Franco Rojas

Gabriel Felipe Bravo Delgadillo

Samuel Eduardo Garcia Corredor

Descripción:

El CNC-BOT DIBUJANTE es un robot que utiliza servo motores digitales y un controlador ESP32 WROOM para mover un lapiz o un boligrafo en dos ejes (X E Y), permitiendo dibujar imagenes con precision. El sistema incluye:

Estructura: Hecha en MDF y Acrilico

Motores: Utiliza los servo motores digitales ADARF-155

Controlador: Utiliza el controlador ESP 32 WROOM V1

Software: Utilizara software diseñado en micropython para interpretar y realizar los movimientos correspondiente

Elementos de dibujo: Puede usar un lapiz o un boligrafo para trazar

El robot sigue una secuencia de instrucciones almacenadas en el controlador, moviendo el lápiz a las coordenadas especificadas para crear el dibujo.

Este proyecto es ideal para quienes desean explorar la intersección entre arte y tecnología, y aprender sobre automatización y control numérico.
